# Strategy Test: Complete Visual Guide

## 1. HOW THE SCORING WORKS

### Per-Choice Grading System

```
SCENARIO 1: RESOURCE ALLOCATION
┌──────────────────────────────────────────────┐
│ You choose: "40% product, 30% team,          │
│             20% marketing, 10% legal"         │
└──────────────────────────────────────────────┘
           ↓ SYSTEM EVALUATES ↓

     ┌─────────────────┐
     │ EFFICIENCY: 80  │  How well you allocate resources
     ├─────────────────┤
     │ FORESIGHT: 85   │  Long-term thinking
     ├─────────────────┤
     │ BALANCE: 85     │  Managing competing interests
     └─────────────────┘
           ↓ CALCULATION ↓
     
     (80 + 85 + 85) / 3 = 83.33 → ROUNDS TO 83
     
┌──────────────────────────────────────────────┐
│ CHOICE SCORE: 83/100                         │
│ FEEDBACK: "Balanced allocation. Team         │
│ stability, legal compliance, market          │
│ awareness."                                   │
└──────────────────────────────────────────────┘
```

### Running Total Across 6 Scenarios

```
┌────────────────────────────────────────────────────────┐
│ SCENARIO 1: Resource Allocation       → Score: 83     │
│ Running Total: 0 + 83 = 83                             │
├────────────────────────────────────────────────────────┤
│ SCENARIO 2: Team Conflict             → Score: 88     │
│ Running Total: 83 + 88 = 171                           │
├────────────────────────────────────────────────────────┤
│ SCENARIO 3: Crisis Triage             → Score: 85     │
│ Running Total: 171 + 85 = 256                          │
├────────────────────────────────────────────────────────┤
│ SCENARIO 4: Partnership Decision      → Score: 87     │
│ Running Total: 256 + 87 = 343                          │
├────────────────────────────────────────────────────────┤
│ SCENARIO 5: Talent Retention          → Score: 90     │
│ Running Total: 343 + 90 = 433                          │
├────────────────────────────────────────────────────────┤
│ SCENARIO 6: Market Pivot              → Score: 85     │
│ Running Total: 433 + 85 = 518                          │
└────────────────────────────────────────────────────────┘

                    ↓ FINAL CALCULATION ↓

            Total Score / Number of Scenarios
                    518 / 6 = 86.33
                    
                FINAL SCORE: 86/100 ✅
```

## 2. THE THREE SCORING DIMENSIONS EXPLAINED

### EFFICIENCY (Resource & Time Management)
```
❌ WEAK (60-70):
   - Reactive choices
   - Doesn't plan ahead
   - Wastes resources
   
⚠️  MODERATE (75-80):
   - Gets the job done
   - Some planning
   - Acceptable resource use
   
✅ STRONG (85-90):
   - Smart resource allocation
   - Maximizes output
   - Minimal waste
```

### FORESIGHT (Long-term Thinking)
```
❌ WEAK (40-50):
   - Only thinks about immediate term
   - Misses hidden consequences
   - Gets blindsided
   
⚠️  MODERATE (70-75):
   - Thinks 6-12 months ahead
   - Catches most risks
   - Some surprises still happen
   
✅ STRONG (85-90):
   - Multi-year thinking
   - Anticipates compound effects
   - Prepared for contingencies
```

### BALANCE (Competing Interests)
```
❌ WEAK (50-60):
   - All-or-nothing approach
   - Ignores stakeholder concerns
   - Creates friction
   
⚠️  MODERATE (70-75):
   - Acknowledges multiple interests
   - Some trade-offs needed
   - Not everyone's happy
   
✅ STRONG (85-90):
   - Finds elegant compromises
   - Everyone wins something
   - Builds consensus
```

## 5. QUICK REFERENCE: WHAT HAPPENS WHEN YOU COMPLETE SCENARIO 6

```
YOU CLICK: "COMPLETE ANALYSIS" Button
              ↓
         handleNextScenario() called
              ↓
    scenarioIndex (5) >= scenarios.length - 1 (5)?
              ↓ YES ↓
         const avgScore = Math.round(518 / 6)
                  = Math.round(86.33)
                  = 86
              ↓
    onComplete({ 
        strategyScore: 86, 
        history: [6 scenarios with scores] 
    })
              ↓
       Callback fires to handleStepComplete()
              ↓
    stepIndex: 16, CALIBRATION_BENCHMARKS.length: 17
    16 < 17? YES
              ↓
       stepIndex < length - 1? NO (16 is not < 16)
              ↓
    ❌ FALSE CONDITION: DON'T move to next step
    ✅ TRUE CONDITION: Call finalizeOnboarding()
              ↓
         All 17 tests are now collected
    Data flows to dossier generation
              ↓
      Report is set → ClassifiedDossier renders
              ↓
    User sees their complete calibration profile
              ↓
    User clicks "PROCEED" → Game begins! 🎮
```
