## Overview
The Genesis Protocol implements a **rank-up system inspired by Solo Leveling**, where users progress through 8 ranks (E â†’ D â†’ C â†’ B â†’ A â†’ S â†’ SS â†’ SS+) based on their **Apex Threat Index (HATI)**, which is calculated from their stats.

### 3. **Progression Timeline is Too Fast**
Currently, users can rank up very quickly:
- Each stat baseline is **low** (0 at start)
- HATI calculation aggregates across 6 stats
- Reaching 60% (B-rank) threshold is achievable in weeks, not months

**Problem Scenario**:
- User completes ~10-15 daily tasks â†’ +150-300 XP/day
- At ~200 XP/day, reaching D-rank (20%) takes ~2-4 weeks (realistic)
- But reaching C-rank (40%) takes another ~4-8 weeks
- And reaching B-rank (60%) might only take 2-3 months total

**This is too fast.** Real-world skill development (chess, fitness, martial arts) takes much longer.

## Directive System Design

### **What is a Directive?**
A **Directive** (Promotion Exam) is a **customized, actionable challenge** that:
1. **Is specific to the target rank** (not generic)
2. **Tests mastery in key areas** (e.g., physical, mental, spiritual)
3. **Has multiple objectives** (e.g., "Run 5km", "Meditate 30min", "Read a book")
4. **Must be completed to rank up**
5. **Is generated by Central** (via Gemini AI with secure backend)

### **Example Directives by Rank**

#### **Eâ†’D: "Initiation Protocol"**
- Objectives:
  - [ ] Complete 10 daily tasks
  - [ ] Journal 3 entries about your goals
  - [ ] Do 1 week of consistent habits
  - [ ] Answer the onboarding questionnaire

#### **Dâ†’C: "Foundation Assessment"**
- Objectives:

#### **Câ†’B: "Elite Candidacy"**

  - [ ] Complete 50 daily tasks

  - [ ] Complete Chapter Black story progression

  { secrets: [geminiApiKey] },

          res.status(500).json({ error: 'Missing GEMINI_API_KEY' });
          return;

        const body = (req.body || {}) as Partial<GeneratePromotionExamRequest>;
        const userName = String(body.userName || 'Operative');
        const currentRank = String(body.currentRank || 'E');
        const targetRank = String(body.targetRank || 'D');

        const ai = new GoogleGenAI({ apiKey });

Current Rank: ${currentRank} | Target Rank: ${targetRank}

Create a rank-up challenge JSON with:

}

- Rank-appropriate difficulty
- Achievable in 2-4 weeks

- Clear pass/fail criteria

Return ONLY valid JSON.`;

        const response = await ai.models.generateContent({
          model,

export const generatePromotionExam = async (gameState: GameState): Promise<{ mission: Mission, exam: PromotionExam }> => {
    const currentRankIndex = RANKS.findIndex(r => r.name === gameState.rank.name);
    const nextRank = currentRankIndex < RANKS.length - 1 ? RANKS[currentRankIndex + 1] : RANKS[RANKS.length - 1];
    
    const response = await fetch(
        `${process.env.REACT_APP_BACKEND_URL || 'http://localhost:5001/genesis-protocol-staging/us-central1'}/generatePromotionExamV2`,
        {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({
                userName: gameState.userName,
                currentRank: gameState.rank.name,
                targetRank: nextRank.name
            })
        }
    );
    
    if (!response.ok) throw new Error('[Gemini Service] Promotion exam generation failed');
    return response.json();
};
