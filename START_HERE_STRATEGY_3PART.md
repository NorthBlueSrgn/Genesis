## 🎮 The 3-Part Assessment

### Part 1: WARM-UP 🟡
```
🌊 River Crossing (Non-Chess Logic Puzzle)

Time:        1–2 min
Points:      0–30
Tests:       Forward planning, contingency thinking
Example:     "Transport wolf, goat, cabbage across river"
Key Insight: Must bring goat BACK (Plan B thinking)
Scoring:     Optimal (25–30) → Good (15–24) → Partial (5–14) → Wrong (0–4)
Bonus:       +5–10 for explaining contingencies
```

### Part 2: CORE 🟢
```
♔ Chess Tactics (2 Puzzles)

Time:        2–4 min
Points:      0–50 (~25 per puzzle)
Tests:       Tactical calculation, pattern recognition
Puzzle 1:    "Material Gain" → Move: Qh7 (queen sacrifice)
Puzzle 2:    "Tactical Fork" → Move: Nf7 (knight fork)
Input:       Algebraic notation (Qh7, Nxe5, O-O, etc.)
Flow:        Move → Feedback → Reflection
Bonus:       +10–15 for verbal depth, +5–10 for speed <2 min
```

### Part 3: CLOSER 🟣
```
📊 Resource Prioritization (Real-World Scenario)

Time:        1–2 min
Points:      0–20
Tests:       Decision-making under constraints, risk framing
Scenario:    4 project tasks (Bug fix, Feature, Research, Competitor)
Input:       Rank all 4 → Justify → Provide contingency
Optimal:     A (bug) first, then C (research), B (feature), D (analysis)
Scoring:     Correct priority (15–20) → Good (10–14) → Adequate (5–9) → Weak (0–4)
Bonus:       +5 for risk language ("critical", "impact", "if/then")
```

## 📊 SCORING SYSTEM

### Raw Points per Part
```
Part 1 (River):      0–30 pts
Part 2 (Chess):      0–50 pts ← Most heavily weighted
Part 3 (Resource):   0–20 pts
─────────────────────────────
TOTAL:               0–100 pts
FINAL SCORE:         Average of 3 parts (0–100)
```

### Example: Perfect Run
```
Part 1: 28/30 (OPTIMAL) = 93%
Part 2: 50/50 (OPTIMAL) = 100%
Part 3: 20/20 (OPTIMAL) = 100%

Final Score: (93 + 100 + 100) / 3 = 98/100 ✅
```

### Substat Contributions (0–95 each)
```
Strategy:    avg(Part1 + Part2) — Primary strategic foresight
Reason:      avg(Part2 + Part3) — Logical decision-making
Perception:  Part2 + speed bonus — Pattern recognition
Conviction:  avg(Part1 + Part3 depth) — Decision confidence
Resilience:  Contingency depth — Plan B thinking

→ All feed into INTELLIGENCE stat calculation
```
