# C-Tier Baseline Calibration – Quick Reference

## The Framework

**C-Tier = 50%** represents an **average person** across all substats.

From this anchor point:
- **Easy substats** → Achievable B/A (70-85%) with moderate effort
- **Moderate substats** → Requires training/study to reach A (75-80%)
- **Hard substats** → Rare to reach S (85%+), requires extraordinary discipline
- **Variable substats** → No "average"; depends on culture/personality (5-95% valid)

## Expected Ranges (50% Baseline)

### If Testing ONE Strong Area (e.g., Physical = 80%)
```
Physical stat: 78-82% ← Boosted
Vitality stat: 48-52% ← Normal
Psyche stat:   48-52% ← Normal
Spirit stat:   48-52% ← Normal
Intelligence:  48-52% ← Normal
Creativity:    48-52% ← Normal
```

**Key:** Only Physical boosted, no spillover.

---

### If Testing MULTIPLE Strong Areas (e.g., Physical 80%, Intelligence 75%)
```
Physical:      75-80%
Intelligence:  70-75%
Others:        45-52%
```

**Key:** Both boosted proportionally, others unaffected.

---

### If Testing HARD Substat (e.g., Willpower 90%)
```
Psyche stat:   80-88% (boosted, but not proportionally)
Focus:         80-90% (hard substat, rare)
Willpower:     80-90% (hard substat, rare)
Other Psyche:  50-60% (dragged down by weak areas)
Other stats:   48-52% (unchanged)
```

**Key:** Hard substats don't inflate linearly; capped at 95%.

---

### If Testing EASY Substat (e.g., Expression 90%)
```
Creativity:    55-65% ← Boosted, but not to A (limited by other 4 substats at 50%)
Expression:    90% ← Easy to achieve
Imagination:   50% ← Unaffected
Others:        48-52% ← Unchanged
```

**Key:** Easy substats boost their stat, but not excessively (balanced by harder substats).
