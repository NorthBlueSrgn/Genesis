# 🎮 HATI Progression System - Solo Leveling Style

## Overview

The **HATI (Human Apex Threat Index)** is Genesis Protocol's core progression system, inspired by Solo Leveling's rank-based hunter system. It measures your overall capability across all domains and determines your rank within the Ascendant hierarchy.

---

## 📊 HATI Calculation Formula

HATI is calculated using a **three-part weighted system**:

```
HATI = (Base Stats × 40%) + (Mission XP × 40%) + (Bonuses × 20%)
```

### 1️⃣ Base Stats (40%)
Weighted average of all 30 substats across 6 main attributes:

| **Attribute**   | **Weight** | **Substats**                                                      |
|-----------------|-----------|-------------------------------------------------------------------|
| Intelligence    | 20%       | Reason, Knowledge, Adaptability, Strategy, Perception             |
| Psyche          | 20%       | Willpower, Resilience, Focus, Conviction, Composure               |
| Creativity      | 18%       | Imagination, Innovation, Style, Expression, Vision                |
| Physical        | 15%       | Strength, Speed, Endurance, Dexterity, Agility, Stamina           |
| Spirit          | 15%       | Faith, Purpose, Empathy, Charisma, Tranquility                    |
| Vitality        | 12%       | Balance, Adherence, Longevity, Regeneration, Stamina              |

**Formula:**
```
Base Stats Score = (Intelligence × 0.20) + (Psyche × 0.20) + 
                   (Creativity × 0.18) + (Physical × 0.15) + 
                   (Spirit × 0.15) + (Vitality × 0.12)
```

### 2️⃣ Mission XP (40%)
Experience gained from completing missions, tasks, and challenges.

- **Scaling:** 1000 XP = 10 percentile points
- **Expected Gains:** ~100 XP/day = 30% HATI increase over 3 months
- **Conversion:** XP → Percentile → HATI contribution (capped at 100)

**Example:**
- Daily engagement: 100 XP/day
- Monthly: 3,000 XP = 30 percentile points
- 3-month contribution: ~12% to total HATI (30% × 0.40)

### 3️⃣ Bonuses (20%)
Synergies, achievements, and special conditions.

**Synergy Bonuses:**
| **Synergy**                | **Condition**                     | **Bonus** |
|----------------------------|-----------------------------------|-----------|
| Strategic Athlete          | Physical 80+ & Intelligence 80+   | +3%       |
| Visionary Technologist     | Creativity 80+ & Intelligence 80+ | +4%       |
| Uncompromising Creator     | Willpower 80+ & Innovation 80+    | +3.5%     |
| Apex Threat                | 3+ vectors at 70+                 | +5%       |

**Achievement Bonuses** (future implementation):
- First S-rank stat: +2%
- Complete all onboarding: +1%
- 30-day streak: +2%
- Elite performance (top 5%): +3%

**Max Bonus Cap:** 20%

## 🔧 Technical Details

### HATI Calculation Code
```typescript
// Base Stats (40%)
const baseStatsScore = (
  physicalAvg * 0.15 +
  vitalityAvg * 0.12 +
  intelligenceAvg * 0.20 +
  creativityAvg * 0.18 +
  spiritAvg * 0.15 +
  psycheAvg * 0.20
);

// Mission XP (40%)
const missionXPScore = Math.min(100, missionXP / 10);

// Bonuses (20%)
bonusScore = synergies + achievements; // Max 20%

// Final HATI
let hati = (baseStatsScore * 0.40) + (missionXPScore * 0.40) + bonusScore;

// Baseline Adjustment
if (missionXP === 0 && hati < 60) {
  hati = 60; // B-Rank floor
  baselineAdjusted = true;
}
```

### Rank Determination
```typescript
function getHATIRank(hati: number): string {
  if (hati >= 99.9) return 'SS+';
  if (hati >= 97) return 'SS';
  if (hati >= 90) return 'S';
  if (hati >= 75) return 'A';
  if (hati >= 60) return 'B';
  if (hati >= 40) return 'C';
  if (hati >= 20) return 'D';
  return 'E';
}
```
