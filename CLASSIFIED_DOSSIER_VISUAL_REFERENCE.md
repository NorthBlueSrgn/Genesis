## Color Legend

| Color | Usage | Meaning |
|-------|-------|---------|
| Purple (`#a855f7`) | Primary accents, tier 01-02E | Core classification system |
| Cyan (`#06b6d4`) | Tier 03, secondary emphasis | Technical/analytical data |
| Red (`#dc2626`) | Warnings, failure nodes, stamps | Critical or attention-required |
| Green (`#4ade80`) | Biometric data, success metrics | Physical/positive indicators |
| Orange (`#f97316`) | Caution/risk, obsession level | Medium-severity alerts |
| Amber (`#fbbf24`) | Reference points, golden metrics | Premium/legendary items |
| White | Final tier, section separators | Ultimate authority/completion |
| Gray (`#999` - `#666`) | Neutral/background text | Less critical information |

## Visual Effects Applied

### Across All Sections
- **Film Grain**: Subtle noise (opacity 3-8%)
- **Scanlines**: Horizontal drift animation
- **Grid Background**: Subtle 40px purple grid
- **Vignette**: Darkened edges for focus

### Header
- **Chromatic Aberration**: Red/cyan text shadow
- **Animated Scanline**: Biometric scan effect
- **Classified Stamp**: Rotating red wavy stamp with pulse

### Cards/Panels
- **Border Colors**: Match section tier color
- **Hover Effects**: Border lightens, may scale
- **Hover Icons**: Opacity increases on background icon
- **Rounded Corners**: Minimal (2px) for technical feel

### Text
- **Headers**: Orbitron font, tracking 0.2em-0.5em
- **Code-like**: JetBrains Mono for metrics
- **Handwritten**: Permanent Marker for annotations
- **Emphasis**: Bold/italic with color tints

## Interactive Elements

- **Scroll to Unlock**: Scroll reveals sections with staggered animations
- **Clearance Levels**: 
  - Scroll > 10%: Level 1 unlock
  - Scroll > 40%: Level 2 unlock
  - Scroll > 70%: Level 3 unlock
- **Bottom Hint**: Chevron icon bounces to encourage scrolling (disappears at level 3)
- **Hover Cards**: Scale slightly (1.05x) on hover
- **Proceed Button**: Glowing purple with animated shine effect


## Responsive Behavior

- **Mobile**: Single column, larger tap targets
- **Tablet**: Two-column grids where applicable
- **Desktop**: Full multi-column layouts with side panels
- **Header**: Sticky, adjusts logo/info display based on screen width

## Animation Sequence

1. **Mount** (0-2.8s): Biometric scan animation
2. **Reveal** (2.8s+): Dossier fades in, sections slide from bottom
3. **Stagger Delays**:
   - Section 1: 0ms
   - Section 2A: 100ms
   - Section 2B: 100ms
   - Section 2C: 100ms
   - Section 2D: 100ms
   - Section 2E: 100ms
   - Section 3: 200ms
   - Section 4: 300ms
   - Section 5: 400ms
   - Section 6: 500ms
   - Section 7: 600ms
4. **Continuous**: Scanlines drift, stamps pulse, text flickers

---

This visual structure creates an immersive classified intelligence document experience while maintaining information clarity and user engagement through progressive disclosure and cyberpunk aesthetics.
